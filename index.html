Research & Projects

Autonomous Robotics – Grand Challenge

Developed a mobile robot capable of autonomously navigating cluttered environments and performing pick-and-place operations, successfully collecting 4 out of 6 colored blocks.

Implemented advanced robotics algorithms, including image processing for object recognition, motion control, and localization.

May 2024

Improved RRT-Connect Algorithm for Path Planning

Enhanced RRT-Connect with target-biased sampling and adaptive step length to accelerate exploration and improve path efficiency.

Applied a shortcutting technique to smooth final paths, reducing abrupt direction changes and improving overall path quality.

Achieved a 46.45% performance improvement in speed and optimality compared to standard RRT-Connect.

May 2024

Perception Challenge on Turtlebot Waffle-Pi

Successfully completed all four project tasks: projective geometry, homography-based lane following, object detection, and dynamic obstacle handling.

Implemented vision-based navigation using the robot's camera, overlaying vanishing points and horizon lines for accurate spatial understanding.

Developed real-time stop sign detection and optical flow techniques for dynamic obstacle recognition, ensuring safe and efficient robot motion.

December 2023

ROS-based Object Detection and Navigation with ArUCo Markers

Created ROS nodes, publishers and subscribers, and implemented frame broadcasting and listening techniques to navigate through 5 distinct poses with 90% accuracy in pose estimation.

Retrieved appropriate parameters from ArUCo markers, detected various parts using logical cameras, and used action clients to execute sequential tasks, enabling the robot to navigate through multiple poses in a predefined order.
